## What Each Folder Does (Now Explained)

### Backup Folder
**Purpose:** Store complete history of backups
**Content:** `FollowFlow_Snapshot_2026-04-15_1430.xlsx`
**Update:** New file created each sync (never overwrites)
**Use:** Recovery, historical access, archiving
**Example Location:** Documents/Backups

### Downloads Folder
**Purpose:** Quick access current working file
**Content:** `FollowFlow_Leads_Current.xlsx`
**Update:** Gets replaced each sync (or creates new if user chooses)
**Use:** Quick access, sharing, portability
**Example Location:** Downloads
